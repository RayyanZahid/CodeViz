    2. `packages/server/src/index.ts`:
       - Import Fastify from 'fastify'
       - Import `healthPlugin` from './plugins/health.js'
       - Import `db` from './db/connection.js' (this triggers DB connection + WAL mode setup)
       - Create Fastify instance with logger enabled (pino-pretty transport for development: `{ transport: { target: 'pino-pretty', options: { colorize: true } } }`)
       - Register healthPlugin
       - Listen on port 3100, host '0.0.0.0'
       - Wrap in try/catch: log error and exit(1) on failure
       - Add `pino-pretty` to devDependencies in packages/server/package.json if not already present

    - Run `cd packages/server && npx tsx src/index.ts` — server starts on port 3100
    - In a separate terminal, `curl http://localhost:3100/health` — returns JSON with status 'ok'
    - Stop the server (Ctrl+C)
    - Run `pnpm dev` from root — both server and client start via concurrently
    - Verify server responds at localhost:3100/health while pnpm dev is running
