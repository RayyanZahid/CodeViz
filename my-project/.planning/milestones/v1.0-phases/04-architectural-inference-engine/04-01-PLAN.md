<task type="auto">
  <name>Task 2: Add public graph topology accessors to DependencyGraph</name>
  <files>packages/server/src/graph/DependencyGraph.ts</files>
  <action>
Add the following public methods to the DependencyGraph class. These expose read-only graph topology queries that the inference engine needs, keeping graphlib encapsulated inside DependencyGraph:

1. **getPredecessors(nodeId: string): string[]** — Returns `this.g.predecessors(nodeId) ?? []`. These are nodes that have edges pointing TO nodeId (nodes that import this file).

2. **getSuccessors(nodeId: string): string[]** — Returns `this.g.successors(nodeId) ?? []`. These are nodes that nodeId has edges pointing TO (files this node imports).

3. **getOutDegree(nodeId: string): number** — Returns `this.g.outEdges(nodeId)?.length ?? 0`. Count of outgoing edges.

4. **getInDegree(nodeId: string): number** — Returns `this.g.inEdges(nodeId)?.length ?? 0`. Count of incoming edges.

5. **getOutEdges(nodeId: string): GraphDeltaEdge[]** — Returns outgoing edges with their symbols:
```typescript
return (this.g.outEdges(nodeId) ?? []).map(e => ({
  v: e.v,
  w: e.w,
  symbols: (this.g.edge(e.v, e.w) as EdgeMetadata)?.symbols ?? [],
}));
```

6. **getAllNodeIds(): string[]** — Returns `this.g.nodes()`. Needed for re-evaluating unknown zones.

Place these methods in the "Public API" section after the existing `getNodeMetadata` method. Import `GraphDeltaEdge` from the existing `@archlens/shared/types` import statement (already imported but may need to be added to the type import).

Note: `getInDegree` and `getOutDegree` are already used internally via `this.g.inEdges(id)?.length ?? 0` in `onDeltaComputed`. The new public methods expose these to external callers.
  </action>
