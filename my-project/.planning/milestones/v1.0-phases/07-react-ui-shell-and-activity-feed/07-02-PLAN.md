  <action>
    Create `packages/client/src/panels/NodeInspector.tsx` — the node inspector panel.

    **Props:**
    ```typescript
    interface NodeInspectorProps {
      selectedNodeId: string | null;
      onHighlightNode?: (nodeId: string) => void;
    }
    ```

    **Component structure:**
    ```
    <div> (panel container)
      <div> (header — "Inspector" title + collapse toggle)
      <div> (content — collapsible via CSS max-height transition)
        {selectedNodeId ? <InspectorContent /> : <EmptyState />}
      </div>
    </div>
    ```

    **Panel header:**
    - Text: "Inspector" in uppercase, 11px, monospace, color `#ffffff99`
    - Collapse toggle: `collapsed ? '▶' : '▼'` — local `useState<boolean>(false)`
    - When a node is selected, show node name in header as a small badge
    - `onClick` toggles collapsed state

    **InspectorContent (when selectedNodeId is not null):**

    Read from stores:
    ```typescript
    const node = useGraphStore(s => selectedNodeId ? s.nodes.get(selectedNodeId) : undefined);
    const edges = useGraphStore(s => s.edges);
    const activityFeed = useInferenceStore(s => s.activityFeed);
    ```

    Three sections, each with a small section header (10px, uppercase, monospace, color `#ffffff66`):

    **Section 1: Files**
    - Header: "FILES"
    - List `node.fileList` (from GraphNode type) — each file path in 11px monospace, color `#ffffffaa`, truncated with `textOverflow: 'ellipsis'`
    - If `fileList` is empty, show "No files" in muted text
    - Max 10 files shown; if more, show "+N more" counter

    **Section 2: Dependencies**
    - Header: "DEPENDENCIES"
    - Two sub-groups: "Depends on" (outgoing edges from selected node) and "Depended by" (incoming edges to selected node)
    - Filter `edges` Map: outgoing = edges where `edge.sourceId === selectedNodeId`, incoming = edges where `edge.targetId === selectedNodeId`
    - For each dependency: show the other node's name (look up in `nodes` Map) in 11px monospace. Color based on edge type or just `#ffffffaa`.
    - Show counts in sub-headers: "Depends on (3)" / "Depended by (5)"
    - If `onHighlightNode` provided, clicking a dependency name calls `onHighlightNode(otherNodeId)` to allow cross-panel navigation

    **Section 3: Recent Changes**
    - Header: "RECENT CHANGES"
    - Filter `activityFeed` by `item.event.nodeId === selectedNodeId || item.event.targetNodeId === selectedNodeId`
    - Show last 5 matching items: colored dot (item.iconColor) + sentence (item.sentence) + relative time
    - If no matching events, show "No recent changes" in muted text

    **Empty state (no node selected):**
    - "Click a node to inspect" in 12px, color `#ffffff44`, centered with padding

    **Styling:**
    - All inline styles (project convention)
    - Content scrollable: `overflowY: 'auto'`, `maxHeight: collapsed ? 0 : 400` with CSS transition
    - Section headers have `marginTop: 12` and `marginBottom: 4` for visual separation
    - Background: transparent (sidebar provides background)
    - Panel border-top: `'1px solid rgba(255,255,255,0.08)'`

    Export: `export function NodeInspector({ selectedNodeId, onHighlightNode }: NodeInspectorProps) { ... }`

  <action>
    Create `packages/client/src/panels/RiskPanel.tsx` — the risk panel component.

    **Props:**
    ```typescript
    interface RiskPanelProps {
      onHighlightNode?: (nodeId: string) => void;
    }
    ```

    **Component structure:**
    ```
    <div> (panel container)
      <div> (header — "Risks" title + collapse toggle + unreviewed count badge)
      <div> (content — collapsible)
        {unreviewedRisks.map(risk => <RiskItem />)}
        {reviewedCount > 0 && <ReviewedCounter />}
        {risks.size === 0 && <EmptyState />}
      </div>
    </div>
    ```

    **Panel header:**
    - Text: "Risks" in uppercase, 11px, monospace, color `#ffffff99`
    - Collapse toggle: `collapsed ? '▶' : '▼'` — local `useState<boolean>(false)`
    - Badge showing unreviewed count (number of risks where `reviewed === false`), colored red `#ef4444` background with white text, small rounded pill. Only visible when count > 0. Per user decision: "Badge count on collapsed panel header shows unreviewed risk count."
    - `onClick` toggles collapsed state

    **Store subscription:**
    ```typescript
    const risks = useInferenceStore(s => s.risks);
    const markRiskReviewed = useInferenceStore(s => s.markRiskReviewed);
    ```

    Convert `risks` Map to array, split into unreviewed and reviewed.

    **RiskItem sub-component (for unreviewed risks):**
    - Left: severity indicator — colored circle (10px). Per user decision: red `#ef4444` for `critical`, orange `#f97316` for `warning`, yellow `#eab308` for info (though current types only have `warning` and `critical`).
    - Content: `signal.details` in 12px monospace, color `#ffffffcc`
    - Below details: `signal.type` mapped to readable label (`circular_dependency` → "Circular dependency", `boundary_violation` → "Boundary violation", `excessive_fan_out` → "High fan-out") in 10px, color `#ffffff66`
    - Right side: "Mark reviewed" button (small text button, 10px, color `#ffffff44`, hover `#ffffffaa`). Calls `markRiskReviewed(risk.id)`.
    - Clicking the risk item itself (not the button) calls `onHighlightNode?.(risk.signal.nodeId)` — per user decision: "Clicking a risk highlights affected node(s) on the canvas and pans to them."
    - Bottom border: `1px solid rgba(255,255,255,0.05)` separator

    **ReviewedCounter (per user decision: "Reviewed risks collapse into a '3 reviewed' counter"):**
    - Renders as a single row: `"${reviewedCount} reviewed"` in 11px, color `#ffffff44`
    - Clicking expands to show reviewed risks (local `useState<boolean>(false)` for `showReviewed`)
    - Reviewed risk items render with reduced opacity (`opacity: 0.5`) and no "Mark reviewed" button

    **Empty state:**
    - "No architectural risks detected" in 12px, color `#ffffff44`, centered, with padding

    **Severity color helper (internal):**
    ```typescript
    function severityColor(severity: RiskSeverity): string {
      switch (severity) {
        case 'critical': return '#ef4444';
        case 'warning': return '#f97316';
        default: return '#eab308';
      }
    }
    ```

    **Risk type label helper (internal):**
    ```typescript
    function riskTypeLabel(type: RiskType): string {
      switch (type) {
        case 'circular_dependency': return 'Circular dependency';
        case 'boundary_violation': return 'Boundary violation';
        case 'excessive_fan_out': return 'High fan-out';
        default: return type;
      }
    }
    ```

    **Styling:**
    - All inline styles (project convention)
    - Content: `maxHeight: collapsed ? 0 : 300`, `overflow: 'hidden'`, `transition: 'max-height 0.2s ease'`
    - `overflowY: 'auto'` when expanded
    - Background: transparent
    - Panel border-top: `'1px solid rgba(255,255,255,0.08)'`

    Export: `export function RiskPanel({ onHighlightNode }: RiskPanelProps) { ... }`
