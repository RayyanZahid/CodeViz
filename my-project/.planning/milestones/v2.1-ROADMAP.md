## Overview

Added interactive Inspector Panel — clicking any component node on the canvas opens a detailed sidebar showing the component's files, key exports, and full dependency graph (both incoming and outgoing). Includes smooth pan animation, hover highlights, and wiki-style navigation between components.

## Phases

### Phase 9: Inspector Panel

**Goal**: Users can click any component node to see its full architectural details in a sidebar panel
**Depends on**: Phase 8 (data pipeline delivers correct component-level fields)
**Requirements**: INSP-01, INSP-02, INSP-03, INSP-04, INSP-05, INSP-06
**Plans**: 2 plans

Plans:
- [x] 09-01: Inspector panel React component and Zustand selection state
- [x] 09-02: Wire component data (exports, dependencies in/out) into inspector from graph store

**Details:**
- Rewrote NodeInspector with 4 collapsible sections (Files, Key Exports, Dependencies Out, Dependencies In)
- Zone badge with color mapping (frontend=#3b82f6, api=#8b5cf6, etc.)
- 3-way dismissal: X button, ESC key, empty canvas click
- Dependency aggregation groups edges by target/source, sums dependencyCount
- DependencyRow extracted component with per-row hover highlight
- CountBadge with singular/plural: "(1 import)" vs "(N imports)"
- Self-referencing edge exclusion from both directions
- Smooth Konva.Tween pan animation (0.3s EaseInOut)
- Show N more toggles: 5 for files, 10 for exports
