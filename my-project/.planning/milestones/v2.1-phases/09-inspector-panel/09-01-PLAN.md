**ESC key handler:**
Add a `useEffect` that registers a `keydown` event listener on `document`. When `event.key === 'Escape'` and `selectedNodeId` is not null, call `setSelectedNodeId(null)`. Include `selectedNodeId` in the dependency array so the listener is current. Clean up the listener on unmount.

```typescript
useEffect(() => {
  function handleKeyDown(e: KeyboardEvent) {
    if (e.key === 'Escape' && selectedNodeId) {
      setSelectedNodeId(null);
    }
  }
  document.addEventListener('keydown', handleKeyDown);
  return () => document.removeEventListener('keydown', handleKeyDown);
}, [selectedNodeId]);
```

Pass `onClose={() => setSelectedNodeId(null)}` to the `<NodeInspector>` component. This connects the X button (added in Task 1) to App state.
