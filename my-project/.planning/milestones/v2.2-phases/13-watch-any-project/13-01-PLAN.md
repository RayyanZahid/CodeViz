**watchRoot.ts** ‚Äî Create a new Fastify plugin following the existing plugin pattern (FastifyPluginAsync with typed options):

Options interface:
```typescript
interface WatchRootPluginOptions {
  graph: DependencyGraph;
  inferenceEngine: InferenceEngine;
  aggregator: ComponentAggregator;
  getWatchRoot: () => string;
  setWatchRoot: (dir: string) => Promise<void>;
}
```

The `setWatchRoot` callback is provided by index.ts and encapsulates the full reset sequence.

**GET /api/watch** handler:
- Returns `{ directory: getWatchRoot() }` as JSON with 200 status.

**POST /api/watch** handler:
- Reads `{ directory: string }` from request body.
- Validates that `directory` is a non-empty string. If missing or empty, return 400 with `{ error: "directory is required" }`.
- Uses `fs.access(directory, fs.constants.R_OK)` to check the path exists and is readable. On failure, return 400 with `{ error: "Directory does not exist or is not readable: <path>" }`.
- Uses `fs.stat(directory)` to verify it is actually a directory (not a file). On failure, return 400 with `{ error: "Path is not a directory: <path>" }`.
- Resolves the path to absolute with `path.resolve(directory)`.
- Calls `await setWatchRoot(resolvedPath)` which performs the full reset sequence.
- Returns 200 with `{ directory: resolvedPath }`.

   ```typescript
   async function switchWatchRoot(newDir: string): Promise<void> {
     // 1. Stop current pipeline
     await pipeline.stop();
     // 2. Destroy current inference engine (stops ConfigLoader watcher)
     inferenceEngine.destroy();
     // 3. Remove inference engine's delta listener from graph
     graph.removeAllListeners('delta');
     // 4. Clear in-memory graph
     graph.reset();
     // 5. Purge SQLite tables (graph_nodes, graph_edges)
     db.delete(graphNodes).run();
     db.delete(graphEdges).run();
     // 6. Clear aggregator snapshot cache
     // ComponentAggregator stores lastSnapshot ‚Äî need to add a reset method
     // OR just create a new one. Creating a new aggregator is simpler.
     aggregator = new ComponentAggregator(graph);
     // 7. Broadcast watch_root_changed to all WS clients
     broadcast({ type: 'watch_root_changed', directory: newDir });
     // 8. Update currentWatchRoot
     currentWatchRoot = newDir;
     // 9. Create new InferenceEngine
     inferenceEngine = new InferenceEngine(graph, newDir);
     inferenceEngine.loadPersistedZones();
     // 10. Re-register graph delta and inference event listeners for logging
     // (reuse the same listener code as initial setup)
     registerDeltaLogger();
     registerInferenceLogger();
     // 11. Create and start new Pipeline
     pipeline = new Pipeline(newDir, (batch) => graph.onParseResult(batch));
     await pipeline.start();
     console.log(`[ArchLens] Switched to watching ${newDir}`);
   }
   ```

Add `graph.removeListener('delta', this.deltaHandler)` in InferenceEngine.destroy() ‚Äî but currently the handler is an anonymous arrow function.
