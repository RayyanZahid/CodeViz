<objective>
Build the SnapshotManager class that implements delta-threshold snapshot triggering (INFRA-02) and wire it into the server lifecycle for creation, watch-root switching, and shutdown.

Purpose: Without delta-threshold control, snapshot writes would either happen on every event (unbounded storage growth) or not at all. The SnapshotManager is the gatekeeper that ensures snapshots are captured at meaningful moments -- structural changes immediately, minor changes after accumulation -- with debouncing and FIFO pruning to keep storage bounded.

Output: A working SnapshotManager that subscribes to graph delta events, captures snapshots to SQLite via the repository from Plan 01, broadcasts metadata to clients, and integrates into the server's startup/shutdown/switch lifecycle.
</objective>

**Constants (module-level, UPPER_SNAKE_CASE):**
- `MINOR_THRESHOLD = 10` — minor changes before forced snapshot (per CONTEXT.md locked decision)
- `DEBOUNCE_MS = 3000` — 3 seconds within the 2-5s discretion range
- `MAX_SNAPSHOTS = 200` — FIFO cap targeting ~10-20MB budget (per RESEARCH.md recommendation of 200 over 500 for safer sizing)

**Class: `SnapshotManager`**

Private fields:
- `minorEventCount: number = 0` — accumulator for non-structural changes
- `debounceTimer: ReturnType<typeof setTimeout> | null = null` — debounce timer
- `sequenceNumber: number = 0` — monotonically increasing per session
- `pendingTriggerFiles: string[] = []` — files that triggered pending snapshot

Constructor parameters (all `private readonly`):
- `graph: DependencyGraph`
- `sessionId: string`
- `watchRoot: string`

In the constructor, subscribe to `graph.on('delta', (delta) => this.onDelta(delta))`. Store the bound handler reference so it can be removed in `destroy()`.

**Method: `private onDelta(delta: GraphDelta): void`**

Determine if structural:
```typescript
const isStructural =
  delta.addedNodes.length > 0 ||
  delta.removedNodeIds.length > 0 ||
  delta.addedEdges.length > 0 ||
  delta.removedEdgeIds.length > 0;
```

Also include cycles as structural per RESEARCH.md suggestion:
```typescript
  || delta.cyclesAdded.length > 0
  || delta.cyclesRemoved.length > 0;
```

Collect trigger file IDs from delta: extract unique file paths from `delta.addedNodes` (their id field) and `delta.modifiedNodes` (their id field). Append to `this.pendingTriggerFiles`.

If structural:
- Call `this.scheduleSnapshot()`
- Reset `this.minorEventCount = 0`

If not structural:
- Increment `this.minorEventCount`
- If `this.minorEventCount >= MINOR_THRESHOLD`:
  - Call `this.scheduleSnapshot()`
  - Reset `this.minorEventCount = 0`

**Method: `private scheduleSnapshot(): void`**

If `this.debounceTimer` is not null, `clearTimeout(this.debounceTimer)`.

Set `this.debounceTimer = setTimeout(() => { this.debounceTimer = null; this.captureSnapshot(); }, DEBOUNCE_MS);`

**Method: `captureInitialSnapshot(): void` (public)**

Called explicitly after initial scan completes. Calls `this.captureSnapshot()` directly (no debounce for the initial snapshot).

**Method: `private captureSnapshot(): void`**

1. Get the full graph state: call `this.graph.getSnapshot()` which returns `{ nodes, edges, layoutPositions }`. The existing `DependencyGraph.getSnapshot()` method already provides nodes and edges in wire format.

2. Build the positions record from layoutPositions (already a `Record<string, { x, y, zone }>` from getSnapshot).

3. Build the graph JSON blob:
```typescript
const graphJson = {
  nodes: snapshot.nodes,
  edges: snapshot.edges,
  positions: Object.fromEntries(
    Object.entries(snapshot.layoutPositions).map(([id, pos]) => [id, { x: pos.x, y: pos.y }])
  ),
};
```

4. Build summary string: `"${snapshot.nodes.length} nodes, ${snapshot.edges.length} edges"`

5. Capture trigger files, then reset: `const triggerFiles = [...new Set(this.pendingTriggerFiles)]; this.pendingTriggerFiles = [];`

6. Increment sequence number: `this.sequenceNumber++`

7. Insert via repository:
```typescript
const insertedId = snapshotsRepository.insert({
  sessionId: this.sessionId,
  watchRoot: this.watchRoot,
  sequenceNumber: this.sequenceNumber,
  timestamp: new Date(),
  graphJson,
  summary,
  triggerFiles,
  riskSnapshot: [],
});
```

8. FIFO pruning: check count, delete oldest if over cap:
```typescript
const count = snapshotsRepository.getCount(this.sessionId);
if (count > MAX_SNAPSHOTS) {
  snapshotsRepository.deleteOldest(this.sessionId);
}
```

9. Build SnapshotMeta and broadcast:
```typescript
const meta: SnapshotMeta = {
  id: insertedId,
  sessionId: this.sessionId,
  sequenceNumber: this.sequenceNumber,
  timestamp: Date.now(),
  summary,
  triggerFiles,
};
broadcast({ type: 'snapshot_saved', meta });
```

10. Log: `console.log(\`[Snapshot] #${this.sequenceNumber}: ${summary}\`);`

**Method: `destroy(): void` (public)**

- If debounceTimer is not null, clearTimeout
- Remove the delta listener from graph: `this.graph.removeListener('delta', this.deltaHandler)` (where `deltaHandler` is the bound handler stored in constructor)

**Method: `getSessionId(): string` (public getter)**

Returns `this.sessionId`.
