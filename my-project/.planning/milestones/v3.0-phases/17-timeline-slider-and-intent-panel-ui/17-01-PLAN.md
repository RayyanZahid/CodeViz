**intentStore.ts** — Create a new Zustand slice following the exact double-paren pattern from replayStore.ts and inferenceStore.ts.

Interface fields:
- `activeSession: IntentSession | null` — currently active intent session (null when no activity)
- `intentHistory: IntentSession[]` — closed sessions for history log, newest-first, capped at 50
- `applyIntentUpdated: (session: IntentSession) => void` — sets activeSession; also detects focus shifts: if there was a previous activeSession with a different category, push a focus-shift entry to intentHistory before replacing
- `applyIntentClosed: (sessionId: string, endSnapshotId: number | null) => void` — if activeSession exists and matches sessionId, close it: set endSnapshotId and endedAt (Date.now()), move to intentHistory (prepend, cap 50), set activeSession to null
- `loadHistory: (sessions: IntentSession[]) => void` — seed intentHistory from GET /api/intents response; sort by startedAt descending, cap at 50
- `resetState: () => void` — clear activeSession and intentHistory (called on watch_root_changed)

Import `IntentSession` from `@archlens/shared/types`. Export both `useIntentStore` (React hook) and `intentStore` (vanilla reference, same pattern as graphStore/replayStore).

**replayStore.ts** — Add these NEW fields to the existing ReplayStore interface (do NOT remove existing fields):

New state fields:
- `snapshots: SnapshotMeta[]` — loaded from GET /api/timeline, grows via appendSnapshot
- `currentSnapshotIndex: number` — index of currently displayed snapshot in snapshots array (-1 when not scrubbing)
- `isPlaying: boolean` — auto-playback active
- `playbackSpeed: 0.5 | 1 | 2 | 4` — current speed multiplier
- `diffBaseSnapshotId: number | null` — second snapshot for diff overlay (shift-click)

New actions:
- `setSnapshots: (metas: SnapshotMeta[]) => void` — replace snapshots array
- `appendSnapshot: (meta: SnapshotMeta) => void` — append to snapshots array (called on snapshot_saved WS message, even during replay so the live-edge grows)
- `setCurrentSnapshotIndex: (index: number) => void`
- `setIsPlaying: (playing: boolean) => void`
- `setPlaybackSpeed: (speed: 0.5 | 1 | 2 | 4) => void`
- `setDiffBase: (snapshotId: number | null) => void`

Import `SnapshotMeta` from `@archlens/shared/types`.

Default values: snapshots=[], currentSnapshotIndex=-1, isPlaying=false, playbackSpeed=1, diffBaseSnapshotId=null.

In `exitReplay()`, also reset: currentSnapshotIndex=-1, isPlaying=false, diffBaseSnapshotId=null. Do NOT clear snapshots (they persist across replay sessions). Do NOT clear playbackSpeed (user preference persists).

In `enterReplay()`, set isPlaying=false (entering replay always starts paused).
