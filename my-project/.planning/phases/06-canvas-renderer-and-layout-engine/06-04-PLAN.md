1. **Create canvas/AnimationQueue.ts** — manages glow effects on the animation layer:

   - Import `Konva` from `konva`
   - Import `{ getZoneLayout }` from `../layout/ZoneConfig.js`

   - Constants: `DECAY_MS = 30_000` (30 seconds per CONTEXT.md), `MAX_SHADOW_BLUR = 20`, `MAX_SHADOW_OPACITY = 0.8`

   - Class `AnimationQueue`:
     - Private `animLayer: Konva.Layer`
     - Private `active: Map<string, { glowShape: Konva.Rect; startTime: number; edgeGlows: Konva.Line[] }>` — active glow entries
     - Private `animation: Konva.Animation | null`

     - Constructor(animLayer: Konva.Layer):
       - Store animLayer
       - Create `Konva.Animation` scoped to animLayer:
         ```
         this.animation = new Konva.Animation(() => { this.tick(); }, animLayer);
         ```
       - Start animation (it runs continuously on the anim layer only)
       - IMPORTANT: Do NOT call layer.draw() inside the animation callback — Konva handles it (RESEARCH.md anti-pattern)

     - Method `activate(nodeId: string, nodeShape: Konva.Group, zoneName: string, edgeLines?: Konva.Arrow[])`:
       - If already active: reset startTime (extend glow)
       - Get glow color from `getZoneLayout(zoneName).glowColor`
       - Create a glow overlay shape on the animation layer:
         - Clone the node's rect bounds (position/size) but as a new Konva.Rect on the animation layer
         - Set: shadowEnabled=true, shadowColor=glowColor, shadowBlur=MAX_SHADOW_BLUR, shadowOpacity=MAX_SHADOW_OPACITY, fill='transparent', stroke='transparent', shadowForStrokeEnabled=false, listening=false
         - Position it at the same x/y as the node shape
       - For edge glow: create transparent copies of passed edge lines on the animation layer with shadowEnabled=true and matching glowColor
       - Add to animLayer
       - Store in active map

     - Private `tick()`:
       - For each active entry:
         - Compute elapsed = Date.now() - startTime
         - If elapsed >= DECAY_MS: remove glow shapes from animLayer, delete from active map (per RESEARCH.md Pitfall 4 — remove shapes after decay to keep anim layer lightweight)
         - Else: compute progress = elapsed / DECAY_MS (0 to 1)
           - Set shadowOpacity = MAX_SHADOW_OPACITY * (1 - progress)
           - Set shadowBlur = MAX_SHADOW_BLUR * (1 - progress)
           - Apply same decay to edge glow shapes
       - Per CONTEXT.md: "Static glow + decay — solid bright glow that slowly dims over 30 seconds, no pulsing"

     - Method `activateFromDelta(addedNodeIds: string[], updatedNodeIds: string[], nodeRenderer: NodeRenderer, edgeRenderer: EdgeRenderer, edges: Map<string, GraphEdge>)`:
       - For each nodeId in addedNodeIds and updatedNodeIds:
         - Get node shape from nodeRenderer.getShape(nodeId)
         - Get zone from graphStore node data
         - Get edges where sourceId === nodeId, retrieve their Arrow shapes from edgeRenderer
         - Call activate(nodeId, shape, zone, edgeArrows)
       - Per CONTEXT.md: "All active nodes glow simultaneously during burst activity"

     - Method `destroy()`:
       - Stop animation
       - Remove all glow shapes from animLayer

2. **Create canvas/ViewportController.ts** — zoom/pan/fit-to-view/viewport persistence:

   - Import `Konva` from `konva`
   - Import `{ saveViewport, loadViewport, clearViewport }` from `../utils/viewport.js`
   - Import `{ CANVAS_WIDTH, CANVAS_HEIGHT }` from `../layout/ZoneConfig.js`

   - Constants: `ZOOM_FACTOR = 1.1`, `MIN_SCALE = 0.05`, `MAX_SCALE = 5`

   - Class `ViewportController`:
     - Private `stage: Konva.Stage`
     - Private `onViewportChange: (() => void) | null` — callback for culling/minimap updates

     - Constructor(stage: Konva.Stage, onViewportChange?: () => void):
       - Store stage and callback
       - Attach wheel handler for zoom-to-pointer (per RESEARCH.md Pattern 4):
         ```
         stage.on('wheel', (e) => {
           e.evt.preventDefault();
           const oldScale = stage.scaleX();
           const pointer = stage.getPointerPosition()!;
           const mousePointTo = {
             x: (pointer.x - stage.x()) / oldScale,
             y: (pointer.y - stage.y()) / oldScale,
           };
           const direction = e.evt.deltaY < 0 ? 1 : -1;
           const newScale = clamp(oldScale * (direction > 0 ? ZOOM_FACTOR : 1 / ZOOM_FACTOR), MIN_SCALE, MAX_SCALE);
           stage.scale({ x: newScale, y: newScale });
           stage.position({
             x: pointer.x - mousePointTo.x * newScale,
             y: pointer.y - mousePointTo.y * newScale,
           });
           this.persistViewport();
           this.onViewportChange?.();
         });
         ```
       - Attach dragend handler for pan persistence:
         ```
         stage.on('dragend', () => {
           this.persistViewport();
           this.onViewportChange?.();
         });
         ```
       - Restore viewport from localStorage on init:
         - Call loadViewport()
         - If valid: stage.scale({ x: zoom, y: zoom }), stage.position({ x: panX, y: panY })

     - Method `fitToView(padding = 60)`:
       - Get graphLayer from stage: `stage.findOne('#graph-layer')`
       - Get bounding rect: `graphLayer.getClientRect({ skipTransform: true })`
       - If empty (width=0, height=0): position at canvas center
       - Compute scale: min(stageWidth - 2*padding) / box.width, (stageHeight - 2*padding) / box.height, 1.0)
       - Set stage.scale and stage.position to center the graph
       - Call persistViewport()
       - Call onViewportChange()

     - Method `zoomIn()`: adjust scale by ZOOM_FACTOR, centered on stage center
     - Method `zoomOut()`: adjust scale by 1/ZOOM_FACTOR, centered on stage center
     - Method `resetView()`: clearViewport(), call fitToView()

     - Method `getViewportRect(): { x: number; y: number; width: number; height: number }`:
       - Compute viewport in world coordinates:
         ```
         const scale = stage.scaleX();
         return {
           x: -stage.x() / scale,
           y: -stage.y() / scale,
           width: stage.width() / scale,
           height: stage.height() / scale,
         };
         ```

     - Method `getScale(): number`: return stage.scaleX()

     - Private `persistViewport()`:
       - saveViewport({ zoom: stage.scaleX(), panX: stage.x(), panY: stage.y() })

         ```
         function fullSync(state: GraphStore) {
           nodeRenderer.syncAll(state.nodes);
           placer.placeNewNodes(state.nodes, state.edges);
           // Apply positions to node renderer
           for (const [id, pos] of placer.getPositions()) {
             nodeRenderer.setPosition(id, pos);
           }
           edgeRenderer.syncAll(state.edges);
           cullingIndex.rebuild();
           onViewportChange(); // update culling
           graphLayer.batchDraw();
         }
         ```

       - Subscribe to graphStore for incremental updates:
         ```
         const unsub = graphStore.subscribe((state, prev) => {
           // Detect what changed
           const addedNodeIds = [...state.nodes.keys()].filter(id => !prev.nodes.has(id));
           const removedNodeIds = [...prev.nodes.keys()].filter(id => !state.nodes.has(id));

           // Update renderers
           nodeRenderer.applyDelta(state.nodes, prev.nodes);

           // Remove positions for deleted nodes
           for (const id of removedNodeIds) {
             placer.removeNode(id);
           }

           // Place new nodes
           if (addedNodeIds.length > 0) {
             placer.placeNewNodes(state.nodes, state.edges);
             for (const [id, pos] of placer.getPositions()) {
               nodeRenderer.setPosition(id, pos);
             }
             cullingIndex.rebuild();
           }

           edgeRenderer.applyDelta(state.edges, prev.edges);
           edgeRenderer.updatePositions();

           // Trigger activity glow for added/updated nodes
           const updatedNodeIds = [...state.nodes.keys()].filter(id =>
             prev.nodes.has(id) && state.nodes.get(id) !== prev.nodes.get(id)
           );
           animQueue.activateFromDelta(addedNodeIds, updatedNodeIds, nodeRenderer, edgeRenderer, state.edges);

           onViewportChange(); // update culling
           graphLayer.batchDraw();
         });
         ```
