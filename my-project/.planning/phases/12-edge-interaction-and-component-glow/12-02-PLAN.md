**AnimationQueue.ts changes:**

Enhance the existing glow system to add a 2-3 second pulse phase followed by the existing 30-second linear decay, with a visible bright border on the node's Rect.

1. Add constants:
   ```typescript
   const PULSE_MS = 2_500;      // 2.5 second pulse phase (per CONTEXT.md: 2-3 seconds)
   const PULSE_CYCLES = 3;       // Number of pulse oscillations during PULSE_MS
   ```

2. Update `GlowEntry` interface to track the node shape reference:
   ```typescript
   interface GlowEntry {
     glowShape: Konva.Rect;
     startTime: number;
     edgeGlows: Konva.Line[];
     nodeShape: Konva.Group;    // NEW — reference to the node group for border manipulation
     glowColor: string;         // NEW — stored color for border styling
   }
   ```

3. In `activate()` method:
   - Store `nodeShape` and `glowColor` in the GlowEntry
   - Immediately set a bright border on the node's Rect: `nodeRect.stroke(glowColor)`, `nodeRect.strokeWidth(2.5)`. This makes the change immediately visible on the graph layer.
   - When resetting an already-active entry (the `existing` branch), reset `startTime` to restart the pulse timer (per CONTEXT.md: "each new change resets the pulse timer")

4. Modify the `tick()` method to have TWO phases:

   **Phase 1: Pulse (elapsed < PULSE_MS)**
   - Calculate pulse progress: `const pulseProgress = elapsed / PULSE_MS`
   - Use a sine wave for pulsing: `const pulseValue = Math.sin(pulseProgress * PULSE_CYCLES * Math.PI * 2) * 0.5 + 0.5`
   - Modulate shadow opacity between 40%-100%: `const opacity = MAX_SHADOW_OPACITY * (0.4 + pulseValue * 0.6)`
   - Modulate shadow blur between 50%-100%: `const blur = MAX_SHADOW_BLUR * (0.5 + pulseValue * 0.5)`
   - Apply to glowShape and edge glows (same ratios as current code: edges at 0.5/0.6 of node values)
   - The node border stays at full brightness during pulse phase

   **Phase 2: Decay (elapsed >= PULSE_MS and elapsed < PULSE_MS + DECAY_MS)**
   - Calculate decay progress: `const decayProgress = (elapsed - PULSE_MS) / DECAY_MS`
   - Linear decay of shadow: same as current code but offset by PULSE_MS
   - Linear decay of node border: `nodeRect.stroke(glowColor)` with opacity derived from decayProgress. Use the stored `glowColor` and modulate: construct an rgba version or use Konva's opacity on stroke. Simplest approach: reduce strokeWidth linearly from 2.5 to the node's default (1 or 2 depending on isComponent). Or modulate the stroke color alpha.
   - Better approach for border fade: store the original stroke and strokeWidth on the GlowEntry, then during decay interpolate strokeWidth from 2.5 back to original, and stroke color from glowColor back to original `'rgba(255, 255, 255, 0.25)'`.

   **Phase 3: Complete (elapsed >= PULSE_MS + DECAY_MS)**
   - Restore the node's Rect to its original stroke and strokeWidth (stored on GlowEntry)
   - Destroy glow shapes
   - Remove from active map

5. Store original stroke values when activating:
   ```typescript
   interface GlowEntry {
     // ... existing fields ...
     nodeShape: Konva.Group;
     glowColor: string;
     origStroke: string;
     origStrokeWidth: number;
   }
   ```
   In activate(): `origStroke = nodeRect.stroke()`, `origStrokeWidth = nodeRect.strokeWidth()`

6. In `destroy()`: also restore any active nodes' original stroke/strokeWidth before cleaning up.
