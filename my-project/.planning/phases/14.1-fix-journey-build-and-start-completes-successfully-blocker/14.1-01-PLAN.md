  <files>

    packages/client/tsconfig.json

  <action>
    1. **Root package.json** — Add a `build` script:
       ```json
       "build": "pnpm build:workers && tsc -b packages/shared packages/server packages/client"
       ```
       Insert it after the existing `build:workers` script. This runs the worker build first (server code references worker CJS output), then runs `tsc -b` on all three packages in dependency order.

    2. **packages/shared/tsconfig.json** — Add `"composite": true` to compilerOptions. Do NOT add `outDir` — shared's package.json exports point to source TypeScript directly (`"./types": "./src/types/index.ts"`), and adding outDir would break workspace resolution. The `composite: true` is required for `tsc -b` to accept shared as a project reference target.

    3. **packages/server/tsconfig.json** — Add `"composite": true` to compilerOptions. The server already has `"outDir": "dist"` and `"rootDir": "src"`, so only `composite` is missing. This enables `tsc -b` project reference participation.

    4. **packages/client/tsconfig.json** — Three changes:
       - Add `"composite": true` to compilerOptions
       - Remove `"noEmit": true` (this blocks `tsc -b` from producing JS output; Vite ignores this option anyway since it uses esbuild internally for transpilation)
       - Add `"outDir": "dist"` to compilerOptions (destination for compiled JS)

       The result should be:
       ```json
       {
         "extends": "../../tsconfig.base.json",
         "compilerOptions": {
           "composite": true,
           "module": "ESNext",
           "moduleResolution": "Bundler",
           "lib": ["ES2022", "DOM", "DOM.Iterable"],
           "outDir": "dist",
           "jsx": "react-jsx"
         },
         "include": ["src"]
       }
       ```

    5. **Verify the build works** — Run `pnpm build` from the project root. It should complete without errors. Check that `packages/server/dist/` and `packages/client/dist/` contain `.js` files.
  </action>
  <verify>
    Run `pnpm build` from the project root — must exit 0 with no TypeScript errors.
    Run `ls packages/server/dist/*.js` — must list at least one file.
    Run `ls packages/client/dist/` — must contain files (directory must exist).
  </verify>
  <done>
    `pnpm build` exits 0. Both `packages/server/dist/` and `packages/client/dist/` contain `.js` output files. No TypeScript errors during build.
  </done>

       ```typescript
       import { test, expect } from '@playwright/test';
       import path from 'path';
       import { fileURLToPath } from 'url';

       const __dirname = path.dirname(fileURLToPath(import.meta.url));
       const projectRoot = path.resolve(__dirname, '../../');
       const BASE_URL = process.env.BASE_URL || 'http://localhost:5173';

       test.describe('Build and Start', () => {
         test('App builds without errors', async ({}) => {
           const { execSync } = await import('child_process');
           const result = execSync('pnpm build', {
             encoding: 'utf-8',
             timeout: 120000,
             cwd: projectRoot,
           });
           expect(result).toBeDefined();
         });

         test('Build outputs exist in dist folders', async ({}) => {
           const { existsSync, readdirSync } = await import('fs');
           const serverDist = path.join(projectRoot, 'packages/server/dist');
           const clientDist = path.join(projectRoot, 'packages/client/dist');
           expect(existsSync(serverDist)).toBe(true);
           expect(existsSync(clientDist)).toBe(true);
           const serverJsFiles = readdirSync(serverDist).filter((f: string) => f.endsWith('.js'));
           expect(serverJsFiles.length).toBeGreaterThan(0);
         });

         test('Dev server starts and responds', async ({ page }) => {
           const response = await page.goto(BASE_URL + '/');
           expect(response?.status()).toBeLessThan(400);
           await expect(page.locator('body')).not.toBeEmpty();
         });

         test('No JavaScript errors on load', async ({ page }) => {
           const errors: string[] = [];
           page.on('pageerror', (err) => errors.push(err.message));
           await page.goto(BASE_URL + '/');
           await page.waitForTimeout(2000);
           expect(errors).toEqual([]);
         });
       });
       ```

       - Keep the 2-second wait + zero-JS-errors check as-is (per locked decision)
