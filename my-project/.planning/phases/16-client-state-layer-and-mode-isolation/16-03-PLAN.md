**Function 1: morphNodesToPositions**
```typescript
export function morphNodesToPositions(
  targetPositions: Map<string, { x: number; y: number }>,
  nodeRenderer: NodeRenderer,
  duration = 0.5,
): void
```
- Iterates over `targetPositions` entries.
- For each nodeId, gets the Konva.Group via `nodeRenderer.getShape(nodeId)`.
- If shape exists, creates a `new Konva.Tween` animating `x` and `y` from current `shape.x()` / `shape.y()` to target position.
- Uses `Konva.Easings.EaseInOut`, the given `duration` (default 0.5s = 500ms per CONTEXT.md).
- `onFinish` callback calls `tween.destroy()` for cleanup.
- `tween.play()` to start.
- Use `shape.x()` / `shape.y()` as start point (actual rendered position, NOT placer position — per RESEARCH.md Pitfall 3).

**Function 2: fadeInNodes**
```typescript
export function fadeInNodes(
  nodeIds: string[],
  nodeRenderer: NodeRenderer,
  duration = 0.5,
): void
```
- For each nodeId, get shape via `nodeRenderer.getShape(nodeId)`.
- Set initial `shape.opacity(0)` and `shape.visible(true)`.
- Create Konva.Tween animating `opacity` from 0 to 1, duration, EaseInOut, destroy on finish.

**Function 3: fadeOutNodes**
```typescript
export function fadeOutNodes(
  nodeIds: string[],
  nodeRenderer: NodeRenderer,
  duration = 0.5,
): void
```
- For each nodeId, get shape.
- Create Konva.Tween animating `opacity` from current to 0.
- `onFinish`: set `shape.visible(false)`, `shape.opacity(1)` (restore default), then `tween.destroy()`.

**Function 4: applyReplayTint**
```typescript
export function applyReplayTint(
  nodeRenderer: NodeRenderer,
  tintedFills: Map<string, string>,
): void
```
- Iterates all node shapes via `nodeRenderer.getAllNodeBounds()` to get IDs.
- For each node shape (via `nodeRenderer.getShape(id)`):
  - Find the `Rect` child: `shape.findOne<Konva.Rect>('Rect')`
  - If rect exists, save the current fill in `tintedFills` Map: `tintedFills.set(id, rect.fill())`
  - Blend the fill toward cool blue: compute a blue-shifted color.
    - Simple approach: Use a fixed semi-transparent blue overlay color `'rgba(100, 160, 255, 0.15)'` blended with original.
    - Since exact blending in Konva is complex, use a simpler approach: reduce the rect's fill opacity and add a blue stroke glow.
    - **Recommended implementation:** Save original fill, then set fill to a desaturated/blue-shifted version. For simplicity, if the original fill contains specific zone colors (hex), shift them toward blue:
      - Create a helper `blendToBlue(originalHex: string): string` that mixes the color ~20% toward `#64a0ff`.
      - For each zone color in the palette, pre-compute the blended version.
    - **Simplest robust approach:** Set `rect.shadowColor('#64a0ff')`, `rect.shadowBlur(6)`, `rect.shadowOpacity(0.4)`, `rect.shadowEnabled(true)`. This adds a blue glow around every node without changing the fill color. Store original shadow settings in tintedFills for restore.
    - **CHOSEN APPROACH (per Claude's discretion on tint implementation):** Add a blue shadow glow to each node rect. This is visually distinct, simple to implement, and easy to restore. Store original shadow settings.
  - Store in tintedFills: `tintedFills.set(id, JSON.stringify({ shadowColor: rect.shadowColor(), shadowBlur: rect.shadowBlur(), shadowOpacity: rect.shadowOpacity(), shadowEnabled: rect.shadowEnabled() }))`
  - Apply tint: `rect.shadowColor('#64a0ff')`, `rect.shadowBlur(8)`, `rect.shadowOpacity(0.5)`, `rect.shadowEnabled(true)`

**Function 5: restoreOriginalTint**
```typescript
export function restoreOriginalTint(
  nodeRenderer: NodeRenderer,
  tintedFills: Map<string, string>,
): void
```
- Iterates `tintedFills` entries.
- For each nodeId, get shape and Rect.
- Parse the stored JSON and restore: `rect.shadowColor(original.shadowColor)`, etc.
- Clear tintedFills map.

All five functions are pure (no store imports) — they receive renderer/data as arguments for testability.

**4. Add replayStore subscription for entry/exit transitions:**

Inside the same useEffect (after the graphStore subscription, before the cleanup return), add a replayStore subscription:

```typescript
// Track tinted fills for restore on exit
const tintedFills = new Map<string, string>();
// Track if we need to do exit animation
let wasInReplay = false;

const replayUnsub = replayStore.subscribe((state, prev) => {
  // --- ENTER REPLAY ---
  if (state.isReplay && !prev.isReplay) {
    wasInReplay = true;

    // 1. Compute which nodes exist in both live and historical, which are new, which are removed
    const liveNodeIds = new Set(graphStore.getState().nodes.keys());
    const historicalNodeIds = new Set(state.replayNodes.keys());

    const commonIds = [...liveNodeIds].filter(id => historicalNodeIds.has(id));
    const addedInHistorical = [...historicalNodeIds].filter(id => !liveNodeIds.has(id));
    const removedInHistorical = [...liveNodeIds].filter(id => !historicalNodeIds.has(id));

    // 2. Create shapes for nodes that exist in historical but not live
    //    These need to be added to the renderer temporarily
    for (const id of addedInHistorical) {
      const node = state.replayNodes.get(id);
      if (node) {
        nodeRenderer.createShape(id, node);
      }
    }

    // 3. Compute target positions for historical nodes
    //    Since positions field is {} (Phase 14 deferral), use IncrementalPlacer to compute positions
    //    Clear placer, add only historical nodes, run layout
    const historicalPositions = new Map<string, { x: number; y: number }>();

    // Use placer to compute positions for the historical graph
    placer.placeNewNodes(state.replayNodes, state.replayEdges);
    for (const [id, pos] of placer.getPositions()) {
      if (historicalNodeIds.has(id)) {
        historicalPositions.set(id, pos);
      }
    }

    // 4. Animate common nodes to historical positions (morph)
    const commonPositions = new Map<string, { x: number; y: number }>();
    for (const id of commonIds) {
      const pos = historicalPositions.get(id);
      if (pos) commonPositions.set(id, pos);
    }
    morphNodesToPositions(commonPositions, nodeRenderer, 0.5);

    // 5. Fade in nodes that are historical-only (didn't exist in live)
    //    Position them first, then fade in
    for (const id of addedInHistorical) {
      const pos = historicalPositions.get(id);
      if (pos) nodeRenderer.setPosition(id, pos);
    }
    fadeInNodes(addedInHistorical, nodeRenderer, 0.5);

    // 6. Fade out nodes that are live-only (don't exist in historical)
    fadeOutNodes(removedInHistorical, nodeRenderer, 0.5);

    // 7. Update edges for historical graph
    edgeRenderer.syncAll(state.replayEdges);
    edgeRenderer.updatePositions();

    // 8. Apply cool blue tint to all visible historical nodes
    applyReplayTint(nodeRenderer, tintedFills);

    // 9. Auto-zoom viewport to fit historical graph (~500ms per CONTEXT.md)
    setTimeout(() => {
      viewport.fitToView();
    }, 100);  // Small delay to let morph start before fitting

    gl.batchDraw();
  }

  // --- EXIT REPLAY ---
  if (!state.isReplay && prev.isReplay && wasInReplay) {
    wasInReplay = false;

    // 1. Restore original tint (remove blue glow)
    restoreOriginalTint(nodeRenderer, tintedFills);

    // 2. Re-sync the live graph state onto the canvas
    const liveState = graphStore.getState();

    // Remove any historical-only nodes that were temporarily added
    const historicalNodeIds = new Set(prev.replayNodes.keys());
    const liveNodeIds = new Set(liveState.nodes.keys());
    for (const id of historicalNodeIds) {
      if (!liveNodeIds.has(id)) {
        nodeRenderer.removeShape(id);
      }
    }

    // Full sync to restore live state
    nodeRenderer.syncAll(liveState.nodes);
    placer.placeNewNodes(liveState.nodes, liveState.edges);

    // Morph existing nodes to live positions
    const livePositions = new Map<string, { x: number; y: number }>();
    for (const [id, pos] of placer.getPositions()) {
      livePositions.set(id, pos);
    }
    morphNodesToPositions(livePositions, nodeRenderer, 0.5);

    // Fade in nodes that are live but weren't in historical
    const newInLive = [...liveNodeIds].filter(id => !historicalNodeIds.has(id));
    for (const id of newInLive) {
      const pos = livePositions.get(id);
      if (pos) nodeRenderer.setPosition(id, pos);
    }
    fadeInNodes(newInLive, nodeRenderer, 0.5);

    // Sync edges to live state
    edgeRenderer.syncAll(liveState.edges);
    edgeRenderer.updatePositions();

    // Update nodePositionsRef for cross-panel navigation
    if (nodePositionsRef) {
      nodePositionsRef.current = livePositions;
    }

    // Rebuild culling index
    cullingIndex.setEdges(liveState.edges);
    cullingIndex.rebuild();
    handleViewportChange();

    // Fit to view to show live graph
    setTimeout(() => viewport.fitToView(), 100);

    gl.batchDraw();
  }
});
```

```typescript
/**
 * Fetch a historical snapshot from the server and enter replay mode.
 * Phase 17 timeline slider calls this when the user scrubs to a snapshot.
 * Can also be called programmatically for testing in Phase 16.
 */
export async function loadSnapshotAndEnterReplay(snapshotId: number): Promise<void> {
  const res = await fetch(`/api/snapshot/${snapshotId}`);
  if (!res.ok) throw new Error(`Snapshot ${snapshotId} not found (HTTP ${res.status})`);

  const data = await res.json() as {
    id: number;
    timestamp: number;
    graphJson: {
      nodes: Array<{ id: string; name: string; nodeType: string; zone: string | null; fileList: string[]; incomingEdgeCount: number; outgoingEdgeCount: number; lastModified: string | number; fileCount?: number; keyExports?: string[] }>;
      edges: Array<{ id: string; sourceId: string; targetId: string; edgeType: string; dependencyCount?: number }>;
      positions: Record<string, { x: number; y: number }>;
    };
  };

  // Type imports for GraphNode/GraphEdge are in scope from the imports above
  const nodes = data.graphJson.nodes;
  const edges = data.graphJson.edges;

  replayStore.getState().enterReplay(
    data.id,
    data.timestamp,
    nodes as any,  // Wire format matches GraphNode shape
    edges as any,  // Wire format matches GraphEdge shape
  );
}
```
